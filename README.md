* WebApp for small Dental Clinics. The dentists can manage patients personal and procedures information, billing, costing and a calendar management. The primary goal of this project was to create a tool for my dentist Aunt.
* The projects implements SQLite DB, EF Core, C#, Web API, JS, CSS and HMTL.
